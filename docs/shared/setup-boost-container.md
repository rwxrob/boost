# Setup Linux boost container

For the Boost, we start exploring the Linux operating system and terminal command line interface (CLI) by running it safely in a container (configured in the [Containerfile](../Containerfile)). When combined with a laptop and a good terminal you can explore and learn anywhere — even without an Internet connection. Here's how to get started:

1. [Get admin](get-admin.md) rights on your computer
2. [Install Podman](podman.md) or another OCI container engine
3. [Build, run, remove](build-run-remove-boost-container.md) `boost` container locally
