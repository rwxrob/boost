# Contributing

Please help and contribute any way you can.

Financially [GitHub sponsorship](https://github.com/sponsors/rwxrob) is best, but Twitch cheers, subs, etc. are always welcome.

You can also help by just providing positive comments with useful chapter markers on YouTube videos that need them.

And don't forget you can simply help others in the [discussion forum](https://github.com/rwxrob/boost/discussions) to get searchable answers to their questions.
