# Tech Career Boost

A comprehensive tech career guide for everyone — from first-time learners to veterans leveling up.

AI has changed the game: junior tech roles are vanishing, but there's never been a better time to become a senior-level technologist. Whether you're transitioning careers, accelerating growth, or building as a founder, this Boost shows you what to learn, why it matters, and how to learn it with AI as your partner.

## Choose your path

### New to tech or career switching?

**Start here:** [Beginner Track](beginner/)

You'll go from "should I even try tech?" to building as a technologist at mid-level immediately. No stepping stone junior roles anymore — you need depth from day one. This track covers:

1. Know yourself (saves you hundreds of thousands in wasted education)
2. Set up your lab (Git, GitHub, terminal, Copilot)
3. Build core skills (fundamentals + AI tools)
4. Work like a technologist (as a founder or solo operator)
5. Doctor-level expertise (MCP, agents, advanced patterns)

**Time commitment:** 1 hour/day for 90–120 days to reach job-ready.

**Requirements:** Laptop, GitHub Copilot subscription ($10/month), discipline.

### Already know the basics?

**Jump here:** [Advanced Track](advanced/)

MCP servers, multi-agent orchestration, git worktrees, advanced CI/CD patterns, and cutting-edge practices. For mid-career developers, veterans, and engineers staying current with agentic AI.

## Why this matters

Becoming a technologist used to have rungs: junior → mid → senior. AI has collapsed those rungs. You now go from learning directly to contributing at mid-level. That means:

- **No junior roles** to coast through and build bad habits
- **Depth required immediately** — you must understand yourself, your craft, and how to work with AI from day one
- **Higher pay, faster** — companies pay senior-level salaries because they expect mid-level work immediately
- **Freedom to operate** — work remote as a digital nomad, start a side project, or build solo

This Boost exists to compress that learning curve and teach you the right skills in the right order.

## What you'll learn

- Terminal velocity and how to move fast in development
- Git and GitHub as your version control backbone
- Building with AI partners (GitHub Copilot, Claude, etc.)
- Agentic systems (MCP, agents, multi-step workflows)
- Shipping projects that others can review and that recruiters/clients will see
- Operating like a founder, whether hired or not
