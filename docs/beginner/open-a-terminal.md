# Open a terminal

Everything important happens in the terminal. This is where professionals get things done at "terminal velocity" (a book title I would like to make one day). Here's how to open it on a laptop issued by your company 😉 (Mac or Windows).

Mac: Cmd + Space + "terminal" + Enter  
Win: Super + Space + "terminal" + Enter

## Install Windows Terminal on Windows10

If you still have Windows10 (and not Windows11) you will have to install the _Windows Terminal_ app (previously called _Windows Terminal Preview_).

1. Open Microsoft Store
1. Search for "terminal"
1. Install

Now when you open "terminal" in as above the newly installed Windows Terminal app will open.
